# 🎓 Student Performance Predictor (ML Project)

A simple Machine Learning web application that predicts a student's final marks based on study habits and academic history.

Built using **Python, Flask, and Scikit-learn**.

## 📌 Features

- Predict student performance using ML
- Takes inputs like:
  - Hours Studied
  - Sleep Hours
  - Attendance
  - Previous Marks
- Displays predicted final marks instantly
- Simple and clean web interface

---

## 🧠 Machine Learning Model

- Algorithm: Linear Regression
- Dataset: Custom generated student dataset
- Evaluation Metrics:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - R² Score
